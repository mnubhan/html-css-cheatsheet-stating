Pseudo Selector

Explaination

Is a selector that is used to select specified part of the element based or in specific state.

Type

Psuedo Classes

Explaination

A selector that selects element that are in specific states. It keywords usually start with a colon :.

Syntax

/*button is the element selector , hover is the psuedo class*/

button::hover{
color : blue
}

Table Pseudo Classes

Selector Type Explaination
:fullscreen Element display state Matches an element that is currently in fullscreen mode
:modal Element display state Matches an element that is in a state which it excludes all interaction with elements outside it untill the interaction has been dismmissed

Pseudo Element

Explaination

It select specific part of the element. It keyword usually start with double color ::.

Syntax

div::after {
font-family : Arial, Times, sans-serif
}

Table Pseudo Element

Selector Explaination
::after Create a pseudo-element that is the last child of the selected element.
::before Create a pseudo-element in that is the first child of the selected element
